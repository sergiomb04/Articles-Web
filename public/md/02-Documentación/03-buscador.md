# Buscador Global Instantáneo

Hemos integrado una herramienta de búsqueda que permite encontrar cualquier artículo en milisegundos, indexando títulos, descripciones y etiquetas.

## Cómo Utilizarlo

Tienes dos formas de invocar el buscador:

1. **Icono de Lupa**: En la cabecera de la página (solo en versión escritorio).
2. **Atajo de Teclado**: Presiona `Command + K` (Mac) o `Control + K` (Windows/Linux) en cualquier momento.

## Características de la Búsqueda

- **Instantánea**: Los resultados aparecen mientras escribes.
- **Indexación Completa**: Busca no solo en el título, sino también en las etiquetas y la descripción SEO.
- **Navegación con Teclado**: Una vez abierto, puedes usar las flechas del teclado `↑` `↓` para moverte y `Enter` para ir al artículo.
- **Salida Fácil**: Presiona `Esc` para cerrar el modal rápidamente.

### Ejemplo de uso

Prueba a buscar ahora mismo la palabra "**SEO**" o "**YAML**" usando el atajo `Cmd+K`. Verás cómo aparecen los artículos relacionados de forma inmediata.

> [!NOTE]
> El buscador utiliza un API interno (`/api/search`) que escanea todos los archivos Markdown del servidor en tiempo real.
