# Herramientas y Utilidades

Este artículo describe las herramientas integradas para mejorar la productividad y la experiencia del usuario.

## 1. Botón de Copia
Todos los bloques de código incluyen un botón de copia en la esquina superior derecha con feedback visual.

```javascript
// Haz clic en el botón de arriba a la derecha para copiar este código
function saludar() {
  console.log("¡Hola, mundo!");
}
```

## 2. Tiempo de Lectura Estimado
Al inicio de cada artículo (debajo del título o metadatos), verás un indicador del tiempo estimado de lectura. Este cálculo se basa en la cantidad de palabras del contenido.

> [!NOTE]
> El tiempo de lectura ayuda a los usuarios a gestionar su tiempo y decidir qué leer a continuación.

## 3. Exportación a PDF
En la cabecera de la página, encontrarás un icono de impresora. Al hacer clic, se abrirá el diálogo de impresión del navegador con estilos optimizados:
- Se ocultan elementos innecesarios (header, footer, botones).
- Se expanden los enlaces para mostrar la URL real.
- Los colores se ajustan para un mejor contraste en papel.

Prueba a imprimir este artículo ahora mismo para ver la diferencia.
